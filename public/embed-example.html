Lite Booking Calendar - Embed Example

How to Embed the Calendar

Follow these simple steps to embed the Lite Booking Calendar on your website:

1. Include the embed script

<script src="https://your-domain.com/embed.js"></script>

2. Add a container element

<div id="lite-booking-calendar" data-api="https://your-api-endpoint"></div>

The data-api attribute is optional. If provided, it specifies the API endpoint for fetching calendar data.

3. Optional: Customize the appearance

You can customize the appearance of the calendar by including the custom CSS file before the embed script: 

<link rel="stylesheet" href="https://your-domain.com/calendar-custom.css">
<script src="https://your-domain.com/embed.js"></script>

Then modify the CSS variables in the calendar-custom.css file to match your website's design.

WordPress Integration

For WordPress, you can add the above code to a Custom HTML block or use the provided WordPress plugin for easier integration.
